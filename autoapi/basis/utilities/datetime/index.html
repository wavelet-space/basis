basis.utilities.datetime.parse_date_cs(text: str) → datetime.datetime.date

Parse a Czech date string to datetime.date type.

The name of month must be a valid Czech month name Any case is allowed e.g. ‘prosince’, ‘Prosince’, ‘PROSINEC’ etc.

Parse text:

A date string e.g. ‘1. prosinec 2023’.

Returns:

A date type e.g. ‘(2023, 12, 1)’.
