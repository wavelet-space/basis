Result

Simple monadic result library for Python.

Install

pip install git+https://github.com/wavelet-space/result.git

Usage

import wavelet.result as result

success: Success[int] = Success(1)
print(bool(success))
# True

failure: Failure[int] = Failure(0)
print(bool(failure))
# False
